筆記

pwd

當前資料夾

ls

當前目錄下的資料

-l 長格式

-lh 長格式，人類閱讀

cd

切換資料夾 cd folder/file

man

查看使用說明 man ls

touch

建立新檔案 touch file

rm

刪除檔案 rm file

刪除資料夾rm -rf folder

cp

複製檔案 cp pic.png picCopy.png

pic.png複製目標

picCopy.png複製出來的檔案

cp -R複製整個目錄

mv

移動檔案至資料夾下mv file folder

將檔案移至當前目錄mv folder/file .

將檔案改名mv file01 file02

mkdir

建立資料夾mkdir newFolder

cat

查看檔案cat test.txt
