取得磁碟狀態

工作上總是一堆鳥事，在甲方裡工作有時候又要處理一些完全沒有意義的雜事，浪費時間。

總之因為不明原因，我需要每週紀錄主機的磁碟剩餘容量百分比…

要取得磁碟訊息很簡單，使用System.IO的DriveInfo.GetDrives()可以取得所有磁碟，再把DriveInfo的屬性取出來

string diskStatus = "";
DriveInfo[] allDrives = DriveInfo.GetDrives();

foreach (DriveInfo d in allDrives)
{
    if (d.IsReady == true)//確定磁碟就緒
    {
        int freePercent = (int)(d.TotalFreeSpace / (float)d.TotalSize * 100);
        diskStatus += $"  {d.Name}{freePercent}  "
    }
}

加上錯誤處理

沒什麼意外是不會錯啦，但總是有意外XD，加個try..catch處理就好，最後大概如下。

try
{
    string diskStatus = "";
    DriveInfo[] allDrives = DriveInfo.GetDrives();

    foreach (DriveInfo d in allDrives)
    {
        if (d.IsReady == true)//確定磁碟就緒
        {
            int freePercent = (int)(d.TotalFreeSpace / (float)d.TotalSize * 100);
            diskStatus += $"  {d.Name}{freePercent}  "
        }
    }

    //如果有額外輸入檔案路徑的話，就把第一個參數當作路徑，沒有就丟在桌面
    //若不想在排程器那裡設定開始位置的話，就用絕對路徑吧。
    string path = args.Length == 1 ? args[0] : "DiskLog.txt";
    //記錄時間
    string now = DateRime.Now.ToString("yyyy/MM/dd HH:mm:ss");

    using StreamWriter file = new (path, append: true);
    file.WriteLine($"{now}>> {diskStatus}");
}
catch(Exception ex)
{
    Console.WriteLine(ex);
    Console.ReadKey();//卡個畫面
}

{

    mailMsg.From = new MailAddress("ooo@gmail.com", "主機紀錄機器人");//寄件人地址
    mailMsg.Body = $"{ip} >> {dickStatus}"; //內文

    //設定mail server
    SmtpClient client = new SmtpClient();
    client.Credentials = new System.Net.NetworkCredential("XXX@gmail.com", "****"); //這裡要填正確的帳號跟密碼
    client.Host = "smtp.gmail.com"; //設定smtp Server
    client.Port = 25; //設定Port
    client.EnableSsl = true; //gmail預設開啟驗證
    client.Send(mailMsg); //寄出信件
    client.Dispose();
}
