Hugo指令

新增文章

hugo new post/<md file path>

這樣就可以在content/post/下看到新創的markdown檔

文章調整

基本上我所有的文章都是從jupyter轉換過來
在用hugo new post create好md檔之後 得把jupyter轉換過的md檔複製到 hugo格式中 筆記的位置 需要run 這個指令under root (content folder的上一層)

jupyter nbconvert --to markdown --output-dir=<output dir> <ipynb file>

有一個問題是 轉換後圖會被存在另一個folder中
假設你的jupyter file叫 test.ipynb 在轉換的output dir中還會有一個folder 叫 test_files.ipynb 此時可以把你的圖片都丟到 static/images中 並用以下command line

sed -i '' 's/your_notebook_files\//\/images\//g' your_notebook.md

開幾本地server

hugo server

先在本地測試預覽用的

打包成靜態網頁檔

hugo

hugo -d ../GitHub/MindBreaker3310.github.io

執行完後，打包的靜態網頁會輸出在public/資料夾下

或是給他destination flag輸出到指定資料夾
